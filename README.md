# Roth IRA Calculator

Free, open-source Roth IRA calculator. Plan your retirement savings and see how your investments can grow tax-free.

## [Roth IRA Calculator](https://rothiracalculator.app/)

Features include:
- Projected retirement balance
- Tax-free growth comparison
- Annual contribution calculator
- Early withdrawal penalty estimator

## Quick Calculator

A simplified version is available at: https://wsmhj.github.io/roth-ira-calculator/

## Formula

Future Value = P x ((1+r)^n - 1) / r

Where:
- P = annual contribution
- r = annual return rate
- n = number of years
